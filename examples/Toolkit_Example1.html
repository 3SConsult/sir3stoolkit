Example 1: Shapeimport

This Tutorial demonstrates how to create a connected topological network of nodes and pipes in SIR 3S via Toolkit based on shp data. The data used comes from ‘SIR 3S:nbsphinx-math:Modelle\Beispiele\Wasser\Manual\Projektdaten\01 Shape-Dateien’.
